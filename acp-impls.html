The ANIMAgus Minerva Hermes is an Autonomic Control Plane implementation suitable for Linux kernel environments. It leverages the Linux network namespace mechanisms to provide an ACP that operates below the level of most applications.

It is written in Rust, but leverages RPL written in C++, and IPsec written in C.
